# RiverRaid_2D_Game
2d console terminal game inspired from classic River Raid game
![gameplay](./pics/gameplay.PNG)
![mainMenu](./pics/menu.PNG)

# How to play?
Use arrow keys of your keyboard to control movement of plane and SPACE to fire
